# このシケプリの作り方

ここでは、このシケプリの作り方を紹介しておきます。

Web サイトは、HTML を用いて作るのが一般的です。まずはじめに HTML について紹介します。

HTML を使えば大抵のものを作ることができますが、複雑なものを作るのはあまりにも大変なので、このシケプリではより簡単にドキュメントを作成できる Docusaurus を使いました。そのため、Docusaurus を使った Web サイトの作り方についてその後解説します。HTML についての項と Docusaurus についての項は独立になっているので、HTML の項はひとまず読み飛ばしても構いません。

以降の手順に沿うことで、このシケプリと全く同じようなものを作成することができます。実は、もっと簡単に Web サイトを作成できるサービスはたくさんありアカウント登録だけでできたりしますが、せっかくなので Web 開発者が通常行う方法で解説します。そのため、ただシケプリを作る目的なら少々回り道かもしれません。しかし、この方法を使えばこの先他の言語の環境構築をしようとする時や他のフレームワークを使おうとする時にコマンド一つでできたりするので、あえてこの方法を採用します。

完成品は以下のようになります。

<iframe
  width="100%"
  height="800px"
  style={{ border: "1px solid black" }}
  src="https://docusaurus-sample.onrender.com/docs/intro"
></iframe>

それでは、実際に Web サイトを作っていきましょう。
