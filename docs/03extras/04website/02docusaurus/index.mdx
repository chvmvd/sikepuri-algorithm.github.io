# Docusaurus を用いた Web サイトの作り方

このシケプリは、Docusaurus という静的サイトジェネレーターを使って作られています。
ここでは、Docusaurus を用いた Web サイトの作り方について解説していきます。

Docusaurus では Markdown という非常に簡単な記法でドキュメントを作成することができます。はじめに、Markdown について説明します。

Docusaurus を使うには npm というパッケージ管理ツールが必要です。そのため、まず npm のインストールをします。

次に、npm を用いて Docusaurus のプロジェクトを作成します。

次に、GitHub に Docusaurus のプロジェクトを保存します。

最後に、Render を使ってデプロイします。
