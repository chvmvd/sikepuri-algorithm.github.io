# Docusaurus を用いた Web サイトの作り方

ここでは、Docusaurus を用いた Web サイトの作り方について解説していきます。

このシケプリは、Docusaurus という静的サイトジェネレーターを使っています。Docusaurus を使うには npm というパッケージ管理ツールが必要です。そのため、まず npm のインストールをします。

次に、npm を用いて Docusaurus のプロジェクトを作成します。

次に、GitHub に Docusaurus のプロジェクトを保存します。

最後に、Render にデプロイします。
