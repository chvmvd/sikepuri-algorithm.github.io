# このシケプリの作り方

:::caution
[Visual Studio Code をインストールの項](/docs/03extras/02vscode/)を参考に VSCode をインストールしておいてください。

Windows を使っている場合は、[WSL をインストールの項](/docs/03extras/01wsl/)を参考に WSL もインストールしておいてください。
:::

ここでは、このシケプリの作り方を紹介しておきます。

Web サイトは、HTML を用いて作るのが一般的です。まずはじめに HTML について解説します。

HTML を使えば大抵のものを作ることができますが、複雑なものを作るのはあまりにも大変なので、このシケプリではより簡単にドキュメントを作成できる Docusaurus を使いました。そのため、Docusaurus を使った Web サイトの作り方についてその後解説します。HTML についての項とその後の項は独立になっているので、すでに知っている人は読み飛ばしてください。

以降の手順に沿うことで、このシケプリと全く同じようなものを作成することができます。実は、もっと簡単に Web サイトを作成できるサービスはたくさんありアカウント登録だけでできたりしますが、せっかくなので Web 開発者が通常行う方法で解説します。そのため、少し面倒な設定が多いかもしれません。

完成品は以下のようになります。

<iframe
  width="100%"
  height="400vw"
  src="https://docusaurus-sample.onrender.com/docs/intro"
></iframe>

それでは、実際に Web サイトを作っていきましょう。
