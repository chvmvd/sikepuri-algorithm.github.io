# $\LaTeX$ の導入

なぜか周りに $\LaTeX$ を使っている人が多いので、インストール方法を解説しておきます。

## TeXLive のインストール

[VSCode をインストールの項](/docs/extras/1vscode/)を参考に VSCode をインストールしておいてください。

Windows を使っている場合は、[WSL をインストールの項](/docs/extras/2wsl/)を参考に WSL もインストールしておいてください。

<Tabs groupId="os">
  <TabItem value="win-and-ubuntu" label="Windows 及び Ubuntu">

次の動画の通りにすれば、インストールできます。 VSCode でターミナルを開いている状態にしておいてください。

- ターミナルで次のように入力します。

```shell
sudo apt update && sudo apt -y install texlive-full
```

- パスワードを求められたら、WSL のパスワードを入力します。

- 1 時間もすれば、インストールが自動で完了します。

<video src={installTexLive} controls width="100%" />

:::tip
もし途中でエラーが出てしまったら、もう一度次のコマンドを実行してください。

```shell
sudo apt -y install texlive-full
```

このとき `dpkg was interrupted, you must manually run 'sudo dpkg --configure -a' to correct the problem.` などと表示されたら、言われたとおりに次のコマンドを順に実行してみましょう。大体、これでうまくいきます。

```shell
sudo dpkg --configure -a
```

```shell
sudo apt -y install texlive-full
```

![インストール時のエラー](./tex_error.png)
:::

## $\LaTeX$ を使ってみる

次のようにして、$\LaTeX$ が動くことを確認しましょう。

<video src={typesetTex} controls width="100%" />

- `sample.tex` という名前のファイルを作る。
- `sample.tex` の中に次のように記述する。

```tex
\documentclass{jlreq}
\begin{document}
  サンプルテキスト
  $$
  a^2+b^2=c^2
  $$
\end{document}
```

- ターミナル上で次のコマンドを実行する。

```shell
lualatex sample.tex
```

- `sample.pdf` という名前のファイルができるので、それを開く。

:::tip
Mac を使っている場合は、バックスラッシュ（ `\` ）は、`Alt` を押しながら `¥` を押せば、打つことができます。
:::

:::tip
LuaLaTeX 以外も使うことができます。pLaTeX で jsarticle を使う例も載せておきます。

`sample.tex` の中は次のようにする。

```tex
\documentclass{jsarticle}
\begin{document}
  サンプルテキスト
  $$
  a^2+b^2=c^2
  $$
\end{document}
```

- ターミナル上で次のコマンドを順に実行する。

```shell
platex sample.tex
```

```shell
dvipdfmx sample.dvi
```

:::
