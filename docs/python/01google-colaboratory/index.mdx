# 環境構築（Google Colaboratory）

Google アカウントを用意してログインしておいてください。

ログインした状態でウェブブラウザで <https://colab.research.google.com/> にアクセスすると、次のような画面が表示されるはずです。

`ノートブックを新規作成` をクリックします。

![Google Colaboratory へようこそ](./getting-started.jpg)

新しいファイルが開いたら、コードを書くことができます。

![新規ファイル](./new-file.jpg)

セルの中に `print("Hello World!")` のように書いて、Windows では `Ctrlキー` と `Enterキー` を同時に押すと、Mac では `Cmdキー` と `Enterキー` を同時に押すと、Linux では `Ctrl キー` と `Enterキー` を同時に押すと、任意の Python のコードを実行させることができます。

![実行後](./run.jpg)
