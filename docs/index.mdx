# Introduction to Algorithms

これは、2022 年度アルゴリズム入門のシケプリです。

## 使い方

このシケプリは大きく二部構成になっています。前半で Python の基本を学び、後半で様々なアルゴリズムについて学びます。

授業の内容がある程度わかっていれば、他の教科のテスト対策も忙しいと思うので、Python の基礎の項目は飛ばして様々なアルゴリズムの項目から読んでも良いと思います。その場合、練習問題も飛ばせばすぐに読み終わるかと思います。

授業の内容があまり良くわかっていない人は、はじめから読んでください。かなり丁寧に説明したつもりです。
