# Introduction to Algorithms

これは、アルゴリズム入門のシケプリです。

[Docusaurus](https://docusaurus.io/) を使って作っています。

## 環境構築

```shell
npm ci
```

```shell
pip install pipenv
```

```shell
pipenv sync --dev
```

```shell
pipenv run pre-commit install
```

## 開発

```shell
npm start
```

## ビルド

```shell
npm run build
```

このコマンドは、`build` ディレクトリに静的コンテンツを出力します。
