解答

実は、nnn が素数であるかを判定するのには、n\sqrt{n}n​ までの自然数で割り切れるかを調べれば十分です。これは、もし n\sqrt{n}n​ よりも大きい約数があってもその約数と対になる約数が存在してそれは n\sqrt{n}n​ よりも小さくなるからです。よって、次のようにした方がより高速でしょう。

練習問題 4​

最大公約数を求めるプログラムを作ってみましょう。最大公約数（greatest common divisor）は、GCD とよく略されます。

ヒント

ユークリッドの互除法を用いれば簡単にできます。

ユークリッドの互除法を用いると、次のように最大公約数が計算できるのでした。 gcd(a,b)\mathrm{gcd}(a, b)gcd(a,b) を aaa と bbb の最大公約数とします。

例：303030 と 181818 の最大公約数を求める。

gcd(30,18)=gcd(18,30−18×1)=gcd(18,12)=gcd(12,18−12×1)=gcd(12,6)=gcd(6,12−6×2)=gcd(6,0)=6\begin{aligned} \mathrm{gcd}(30, 18) &= \mathrm{gcd}(18, 30 - 18\times 1) \\ &= \mathrm{gcd}(18, 12) \\ &= \mathrm{gcd}(12, 18 - 12\times 1) \\ &= \mathrm{gcd}(12, 6) \\ &= \mathrm{gcd}(6, 12 - 6\times 2) \\ &= \mathrm{gcd}(6, 0) \\ &= 6 \end{aligned}gcd(30,18)​=gcd(18,30−18×1)=gcd(18,12)=gcd(12,18−12×1)=gcd(12,6)=gcd(6,12−6×2)=gcd(6,0)=6​

よって、最大公約数は 666

解答

ユークリッドの互除法を使うと、最大公約数を求めるプログラムは次のようになります。

a, b = b, a は、次と同じ意味です。

tmp = a
a = b
b = tmp

再帰を使うと次のようにもできます。再帰を使ったプログラムは、後の項で紹介します。

math ライブラリの gcd 関数を使っても計算できます。
