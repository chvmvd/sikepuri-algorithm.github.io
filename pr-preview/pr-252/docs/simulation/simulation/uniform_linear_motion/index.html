差分方程式​

まずは、比較的簡単な等速直線運動から考えていきましょう。

初速度 v0v_0v0​ と ttt 秒後の変位 xxx との関係は次のようになります。

dxdt=v0\frac{dx}{dt} = v_0dtdx​=v0​

この微分方程式は解析的に解けますが、差分化して差分方程式を作ってみましょう。

xn+1−xnΔt=v0∴xn+1−xn=v0Δt∴xn+1=xn+v0Δt\begin{aligned} \frac{x_{n+1} - x_n}{\Delta t} &= v_0 \\ \therefore x_{n + 1} - x_n &= v_0 \Delta t \\ \therefore x_{n + 1} &= x_n + v_0 \Delta t \end{aligned}Δtxn+1​−xn​​∴xn+1​−xn​∴xn+1​​=v0​=v0​Δt=xn​+v0​Δt​
